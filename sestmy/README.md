# School Management PHP + Firebase Scaffold
مشروع تجريبي لإدارة بيانات معلمين وطلاب مرتبط بقاعدة Firebase Realtime Database.
يتضمن:
- تسجيل وتسجيل دخول مبسط.
- صفحات CRUD للطلاب والمعلمين.
- رفع صور إلى مجلد `uploads/` على الخادم.
- مزامنة البيانات مع Firebase Realtime Database عبر REST API.
- تصدير بيانات بصيغة CSV قابلة للفتح في Excel.
- صفحة إحصائيات مبدئية.

**ملاحظة هامة:** هذا مشروع مبدئي يحتاج إلى ضبط قواعد Firebase (قابل للقراءة/الكتابة أو مفاتيح مناسبة) لاختبار العمل بالكامل.
